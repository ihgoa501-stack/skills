# Bullet Journal (子弹笔记)

**Origin:** Ryder Carroll (2013)

**Core idea:** The simplest possible analog productivity system — rapid logging with just a notebook and pen using concise symbols.

## Rapid Logging

### Key Symbols

| Symbol | Meaning | Example |
|--------|---------|---------|
| `·` | Task | `· Review Q3 report` |
| `✕` | Completed | `✕ Review Q3 report` |
| `>` | Migrated (moved forward) | `> Review Q3 report` |
| `<` | Scheduled (in future log) | `< Review Q3 report` |
| `—` | Note | `— Meeting concluded at 4pm` |
| `○` | Event | `○ Sprint review @ 3pm` |
| `*` | Priority | `* Review Q3 report` |
| `!` | Inspiration | `! This approach could work for X` |
| `👁` | Need to explore further | |

### Signifiers
Prepend to bullets for extra meaning:
- `* Important` — `*· Review Q3 report`
- `! Inspiring` — `!— User suggestion for new feature`

## Core Collections

| Collection | Purpose |
|------------|---------|
| **Index** | Table of contents (2–4 pages at front) |
| **Future Log** | 6–12 month overview for non-monthly items |
| **Monthly Log** | Page per month: calendar + task list |
| **Daily Log** | Day-by-day rapid logging |
| **Collections** | Topic-specific pages (e.g., "Books to read," "Project ideas") |

## Migration

Critical ritual: At the end of each month, review open tasks and **migrate** them:
- Still relevant → copy to new month
- Not relevant → strike through (don't carry guilt)
- Scheduled for later → move to Future Log

> **Key insight:** Migration forces you to consciously say "I still care about this" — not just carry dead tasks forward by inertia.

## When to Use
- You prefer pen and paper over apps
- Existing task managers feel too rigid
- You want a fully customizable system
- You value the mindfulness of handwriting
