# North Star Metric (北极星指标)

**Origin:** Sean Ellis, growth hacker — popularized by Amplitude, GrowthHackers, and Silicon Valley product teams

## Core Principle

> **One metric that best captures the core value your product delivers.** Not revenue. Not DAU. The number that *drives* everything else.

## Characteristics of a Good North Star

| Quality | Meaning |
|---------|---------|
| **Measures value to users** | Not "page views" but "messages sent" (WhatsApp) |
| **Leading, not lagging** | Predicts revenue, doesn't just report it |
| **Actionable** | The team can actually move this number |
| **Clear and memorable** | Everyone in the company knows it |
| **Sustains itself** | As it grows, other good things grow too |

## Famous North Stars

| Company | North Star | Why |
|---------|-----------|-----|
| **Airbnb** | Nights booked | The core transaction creating value |
| **Spotify** | Time spent listening | Measures engagement depth, not just app opens |
| **Slack** | Messages sent | The more teams communicate, the more embedded Slack is |
| **Duolingo** | Daily active learners completing a lesson | Not registrations — actual learning |
| **Notion** | Weekly active blocks created | Content creation depth |

## Input Metrics That Drive the North Star

The North Star tells you *where* you are. Input metrics tell you *what to do*:

```
North Star = f(Input A, Input B, Input C)
```

Example: **Messages sent** = f(new users × activation rate × messages per user)

## How to Find Yours

1. **What is your product's "aha" moment?** — The action that makes users say "I get it"
2. **How often should it happen?** — Daily? Weekly? Define the frequency
3. **Does it correlate with retention?** — If it doesn't, it's not a North Star
4. **Can the team explain it in one sentence?** — If not, it's too complicated

## Anti-Patterns

- ❌ **Revenue as North Star** — Revenue is the result, not the driver. (Exception: transactional marketplaces)
- ❌ **Vanity metrics** — "Registered users" (mostly inactive), "Page views" (bots)
- ❌ **Multiple North Stars** — One metric to rule them all. If you have 3, you have 0.
- ❌ **Static North Star** — It should evolve as the product matures

## When to Use

- Setting company/team OKRs — the North Star tells you what the KRs should move
- Aligning cross-functional teams — everyone sees the same number
- Feature prioritization — which feature most directly moves the North Star?
- Fundraising — investors want to know the one number that proves your growth
