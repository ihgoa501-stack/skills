# Skills

AI Agent skills collection — modular, reusable capability extensions for Claude Code and Codex.

## Directory

| Skill | Description |
|-------|-------------|
| [method-gtd](method-gtd/) | GTD — Getting Things Done 任务管理系统 |
| [method-pomodoro](method-pomodoro/) | 番茄工作法 — 专注时间与休息循环 |
| [method-5why](method-5why/) | 5WHY 分析法 — 丰田根本原因追溯 |
| [method-pdca](method-pdca/) | PDCA 循环 — 持续改进 (Plan-Do-Check-Act) |
| [method-okr](method-okr/) | OKR — 目标与关键成果 |
| [method-smart](method-smart/) | SMART 原则 — 目标设定准则 |
| [method-eisenhower-matrix](method-eisenhower-matrix/) | 四象限法则 — 紧急×重要优先级矩阵 |
| [method-agile](method-agile/) | 敏捷开发 — Scrum / Kanban / XP |
| [method-atomic-habits](method-atomic-habits/) | 原子习惯 — 微小改变复合效应 |

## Usage

```bash
# Clone the repo
git clone https://github.com/ihgoa501-stack/skills.git

# Symlink or copy a skill into your Claude skills directory
ln -s "$(pwd)/work-methodology" ~/.claude/skills/work-methodology
```
