# First Principles Thinking (第一性原理)

**Origin:** Aristotle → popularized by Elon Musk

**Core idea:** Strip away assumptions and conventional wisdom. Break a problem down into its most basic, undeniable truths, then rebuild from there.

## How to Apply

```
┌─────────────────────────────────────────────────────────┐
│ 1. Identify current assumption / conventional wisdom    │
│    "Batteries are expensive — that's just how it is"    │
├─────────────────────────────────────────────────────────┤
│ 2. Decompose into fundamental truths                   │
│    What are the raw materials? (Carbon, nickel, etc.)   │
│    What is their market price?                          │
├─────────────────────────────────────────────────────────┤
│ 3. Rebuild from scratch                                 │
│    "If I buy raw materials directly and assemble them,  │
│     can it cost less than buying pre-made batteries?"   │
└─────────────────────────────────────────────────────────┘
```

## Comparison: Reasoning by Analogy vs First Principles

| Analogy | First Principles |
|---------|-----------------|
| "X worked before, so do Y" | "What is fundamentally true?" |
| Faster, but inherits flaws | Slower, but original |
| Incremental improvement | Breakthrough innovation |
| Lower cognitive load | Higher cognitive load |

## Example: SpaceX Rocket Costs

- **Assumption:** Rockets are inherently expensive ($60M+ each)
- **First principles:** Raw materials (aluminum, titanium, copper) cost ~2% of that
- **Result:** SpaceX built Falcon 9 for ~$12M per launch, disrupting the industry

## When to Use
- Stuck on a problem that feels unsolvable
- Everyone says "that's how it's always been done"
- Need breakthrough innovation, not incremental improvement
