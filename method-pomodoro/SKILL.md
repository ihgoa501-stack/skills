# Pomodoro Technique (番茄工作法)

**Origin:** Francesco Cirillo, late 1980s

## Core Rhythm

| Phase | Duration |
|-------|----------|
| Focus session | **25 min** uninterrupted work |
| Short break | **5 min** rest |
| Every 4th cycle | **15–30 min** long break |

## Rules
- A Pomodoro is **indivisible** — once started, must finish
- Interruptions **reset** the current Pomodoro
- Tasks >5–7 Pomodoros should be **split**
- Tasks <1 Pomodoro should be **batched**

## Customizing Intervals

| Work Type | Recommended | Why |
|-----------|-------------|-----|
| Deep creative work | 50 min / 10 min | Longer flow window |
| Admin/busywork | 15 min / 5 min | Low resistance to start |
| Debugging | 45 min / 15 min | Deep focus needed |

## Handling Interruptions

1. **Internal** (your own thought): Write it down on a capture sheet, return to focus
2. **External** (someone interrupts):
   - Inform: "I'm in the middle of something"
   - Negotiate: "Can I get back to you in 25 minutes?"
   - Schedule: If urgent, abort the Pomodoro

## Anti-Patterns
- ❌ Skipping breaks → burnout
- ❌ Using for meetings or creative exploration
- ❌ Not protecting the Pomodoro from interruptions
